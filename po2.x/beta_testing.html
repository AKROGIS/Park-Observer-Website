Beta Testing

You are participating in a beta testing program. Please understand the risks.

  • This app may crash at any time
    Please record what you were doing at the time of the crash and which protocol file you used to create your survey. Send this information to the contact below, so that we can fix the problem.
  • You may lose data
    Emphasis has been placed on ensuring that the data you collect is always saved and safe. However, you may lose data, that is why it is still in Beta. Please let me know if you experience any loss of data.
  • iOS 13+ is required
    The app should work with any version of iOS 13, however all testing to date has been done with iOS 13.5 and higher. If you experience problems, please try upgrade to the latest version of iOS 13. This app makes extensive use of features that are not available for earlier versions of iOS. I'm sorry if you have older devices that will not run iOS 13. You can continue to use the old version of Park Observer or purchase newer devices.
  • Full support for existing files
    Park Observer 2.0 should open and work correctly with any and all files you used with the original Park Observer. This includes surveys (*.poz), maps (*.tpk) and protocols (*.obsprot). If you have an existing file that is not loading, please send it to us so we can fix the problem
  • Minor additions to the protocol file
    Some of the new features require new settings in the protocol file. See the Help pages for details on the protocol file.
  • Tracking issues
    Work continues on some missing features and bugs. If you find a bug, or missing feature please review the list of outstanding issues to see if we are already aware of it. If your issue is not in the list, or you wish to increase the priority of an existing issue, please send your request to the contact below. This list will be updated with each new release.
  • Send your feedback! Feedback is requested and very much appreciated. This app is written for you, and it cannot be all you want without your input. Please send any feedback to the contact below.
