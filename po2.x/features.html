Features

Park Observer 2.0 has numerous improvements over the original Park Observer. The following is a short list of some of the major new features in no particular order.

  • Background track logging.
  • Works on iPhones (and iPads of course).
  • User settings for small, medium and large buttons.
  • User setting to put buttons on top of screen.
  • Supports system settings like larger fonts and dark mode.
  • Alarm control for call in reminders.
  • Angle distance locations are integrated into the attribute form.
  • Locate features with an azimuth and distance.
  • Attribute form is in a side bar not a map popup (no bouncing with keyboard display).
  • More labeling options:
    • Where clause to limit which features are label.
    • Multiple fields in a label
    • Support for arcade expressions (extensive set of functions for formatting feature attributes).
  • Track logging and Transects are both optional
  • Built on the latest frameworks to ensure maximum longevity
